What is currently known about the implimentation of this project:

Spotify, like many other streaming services, requires rich, accurate, and standardized metadata.
DDEX is the standard used by Spotify, Apple Music, Amazon Music, Tidal, Deezer, Qobuz, and Youtube Music. (According to AI overview. Need to verify)
From this website, it looks like it can be automated: https://ddex-suite.org/docs/builder/overview

The requirements for a project like this:
- A website frontend that artists can utilize for managing their music.
- A server backend capable of handling uploads and automated delivery to streaming services.
  - This requres massive ammounts of storage redundancy or cloud solutions.
  - This also requires massive security precautions to protect the server, and the artist from malicious entitiies.
- Payment processing to handle payout to artists
  - This requires a lot of legal knowledge which I just don't have yet.
  - Things such as copyright handling, royalty accounting, and tax reporting
